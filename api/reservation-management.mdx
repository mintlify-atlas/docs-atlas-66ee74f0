## createReservation

Creates a new reservation after validating room availability, dates, and guest data.

### Function Signature

```javascript
function createReservation(hotel, roomNumber, guestData, checkIn, checkOut)
```

### Parameters

<ParamField path="hotel" type="object" required>
  The hotel object where the reservation will be created
</ParamField>

<ParamField path="roomNumber" type="number" required>
  The room number to reserve
</ParamField>

<ParamField path="guestData" type="object" required>
  Guest information object containing:
  
  <Expandable title="Guest Data Fields">
    - `name` (string): Full name of the guest
    - `email` (string): Valid email address (validated with regex pattern)
    - `phone` (string): Phone number in format `+XX XXXXXXXXX` or similar
    - `dni` (string): DNI/ID in format `12345678A` (1-8 digits + letter)
  </Expandable>
</ParamField>

<ParamField path="checkIn" type="string" required>
  Check-in date in ISO format (e.g., '2026-04-10')
</ParamField>

<ParamField path="checkOut" type="string" required>
  Check-out date in ISO format (e.g., '2026-04-15')
</ParamField>

### Validations

The function performs comprehensive validation through `validateReservation()` (validation.js:63-102):

1. **Room Exists**: Verifies the room number exists in the hotel
2. **Room Available**: Checks the room is not reserved for the specified dates
3. **Valid Date Range**: 
   - Check-in must be today or in the future
   - Check-out must be on or after check-in
4. **Maximum Stay**: Cannot exceed 14 nights
5. **Guest Data**: Validates email, phone, and DNI format

### Return Value

<ResponseField name="reservation" type="object | undefined">
  Returns the created reservation object if successful, or `undefined` if validation fails
  
  <Expandable title="Reservation Object Structure">
    <ResponseField name="id" type="string">
      Unique reservation ID in format `RES-{timestamp}-{random}` (generated by utils.js:4-13)
    </ResponseField>
    
    <ResponseField name="roomNumber" type="number">
      The reserved room number
    </ResponseField>
    
    <ResponseField name="guest" type="object">
      Guest information object with name, email, phone, and dni
    </ResponseField>
    
    <ResponseField name="checkIn" type="string">
      Check-in date in ISO string format
    </ResponseField>
    
    <ResponseField name="checkOut" type="string">
      Check-out date in ISO string format
    </ResponseField>
    
    <ResponseField name="nights" type="number">
      Number of nights calculated from check-in to check-out
    </ResponseField>
    
    <ResponseField name="totalPrice" type="number">
      Total price calculated as `pricePerNight * nights`
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Reservation status, initialized as `'confirmed'`
    </ResponseField>
    
    <ResponseField name="extras" type="array">
      Array of extra services, initialized as empty array
    </ResponseField>
  </Expandable>
</ResponseField>

### Error Cases

When validation fails, the function logs the error message and returns `undefined`:

- `"La habitación número {roomNumber} no existe."` - Room doesn't exist
- `"La habitación número {roomNumber} está reservada para esas fechas."` - Room already reserved
- `"El rango de fechas no es válido."` - Invalid date range
- `"Solo se puede reservar hasta 14 noches."` - Exceeds maximum stay
- `"Los datos del huesped no son válidos"` - Invalid guest data

### Example

```javascript
const reservation = createReservation(
  hotel,
  301,
  {
    name: 'Ana García',
    email: 'ana@example.com',
    phone: '+34 612345678',
    dni: '12345678A',
  },
  '2026-04-10',
  '2026-04-15'
)

console.log('Reservation:', reservation)
```

### Expected Output

```javascript
{
  id: 'RES-20260410T120000000Z-42',
  roomNumber: 301,
  guest: {
    name: 'Ana García',
    email: 'ana@example.com',
    phone: '+34 612345678',
    dni: '12345678A'
  },
  checkIn: '2026-04-10',
  checkOut: '2026-04-15',
  nights: 5,
  totalPrice: 500,
  status: 'confirmed',
  extras: []
}
```

<Warning>
  The reservation is immediately added to `hotel.reservations` array when created successfully.
</Warning>

## addExtras

Adds extra services to an existing reservation and recalculates the total price.

### Function Signature

```javascript
function addExtras(hotel, reservationId, extras)
```

### Parameters

<ParamField path="hotel" type="object" required>
  The hotel object containing the reservation
</ParamField>

<ParamField path="reservationId" type="string" required>
  The unique ID of the reservation (e.g., 'RES-001')
</ParamField>

<ParamField path="extras" type="array" required>
  Array of extra service objects. Each extra must contain:
  
  <Expandable title="Extra Object Structure">
    - `name` (string): Name of the extra service (e.g., 'Breakfast', 'Parking')
    - `price` (number): Price per unit/day
    - `quantity` (number): Quantity or number of days
  </Expandable>
</ParamField>

### How Pricing Works

1. **Calculate Extras Price**: For each extra, multiply `price * quantity`
2. **Sum All Extras**: Add all extras prices together (utils.js:122-127)
3. **Update Total Price**: Add extras total to the original room `totalPrice` (utils.js:114-117)

### Return Value

<ResponseField name="reservation" type="object | undefined">
  Returns the updated reservation object with extras added and price recalculated, or `undefined` if reservation doesn't exist
  
  <Expandable title="Updated Reservation">
    <ResponseField name="extras" type="array">
      Array now contains all the added extra objects
    </ResponseField>
    
    <ResponseField name="totalPrice" type="number">
      Recalculated total: original room price + sum of all extras
    </ResponseField>
  </Expandable>
</ResponseField>

### Error Cases

<Warning>
  If the reservation doesn't exist, logs `"La reserva no existe, no se pueden añadir extras"` and returns `undefined`
</Warning>

### Example

```javascript
const updatedReservation = addExtras(hotel, 'RES-001', [
  { name: 'Breakfast', price: 10, quantity: 5 },
  { name: 'Parking', price: 15, quantity: 5 }
])

console.log('Reserva con extras añadidos', updatedReservation)
```

### Expected Output

```javascript
{
  id: 'RES-001',
  roomNumber: 301,
  guest: { /* guest data */ },
  checkIn: '2026-04-10',
  checkOut: '2026-04-15',
  nights: 5,
  totalPrice: 625,  // Original 500 + (10*5) + (15*5) = 625
  status: 'confirmed',
  extras: [
    { name: 'Breakfast', price: 10, quantity: 5 },
    { name: 'Parking', price: 15, quantity: 5 }
  ]
}
```

<Note>
  Extras are appended to the existing `extras` array using spread operator, so you can call this function multiple times to add more extras.
</Note>

### Calculation Example

```javascript
// Original reservation: 5 nights × €100/night = €500
// Breakfast: €10 × 5 days = €50
// Parking: €15 × 5 days = €75
// New total: €500 + €50 + €75 = €625
```
